Challenge 2

Create a new Javascript file called 'course_manager.js' and build a command-line application that assists users with managing their academic progress in five courses.
Functionality:
1. Interactive Course Input: Prompt the user to enter the names of five courses they are taking. Store each entered course name in an appropriate data structure (e.g., an array).
2. Score Input: For each course, request the user to enter their corresponding score.
Finally, your program should print the 5 courses entered and the corresponding scores to the console.

Bonus Challenges:
* Validate the input to ensure it's a valid numerical value (e.g., within a reasonable grade range)
* Calculate and display additional statistics: Include functionalities to calculate and display the average score, highest score, and lowest score across all courses.
